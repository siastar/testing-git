Git Command List

https://www.digitalocean.com/community/tutorials/how-to-use-git-effectively

				  first you need to tell git that you want to use your current directory as a git environment.
				  user@host:~/path/to/project$ git init  
				  output: Initialized empty Git repository in /home/path/to/project/ 

				  Once your have initialized your new empty repository, you can add your files.
				  The following will add all files and directories to your newly created repository: 
				  user@host:~/path/to/project$ git add .  
				  output: No output is good 

				  when you want to "fix" modifies you need to "create a commit"
				  the commit must be marked with a comment which describes the modifies: 
			          user@host:~/path/to/project$ git commit -m "describe what you did" -a  
			          output: (something similar)
			          [master (root-commit) 1b830f8] initial commit 
			          0 files changed
			          create mode 100644 file
				  parameters:
				  -m, expects a commit message like: "describe what you did"
			          -a, applies the commit message to all the modified files
			          -------------
			          evantually it is possible to commit specific files or directories by appending their names like:
			          user@host:~/path/to/project$ git commit -m "describe what you did" file1.js file2.html src ... index.css   

				  at this point we need to create a repository on github:
				  enter your own profile and click on "+" icon in the upper right corner
			          you will be required to chose a repository name, that must be something like
			  	  this-is-the-repository-name and then an optional description.
				  Once the repository is created we may want to push our project on github, to do this we
				  need the link to our repository, there are 2 ways, both from the repo panel in github:
				  1) https link:https://github.com/githubuser/project-name.git
				  2) ssh: git@github.com:githubuser/project-name
