local file management

	 -------------------------------------------------------------------------------------------
	 to initiate a git instance in the project folder
	 $ git init 
	 -------------------------------------------------------------------------------------------
	 to add a single file to the staged area (tracked files list)
	 $ git add filename.xyz
	 -------------------------------------------------------------------------------------------
	 to check what is in the staged area
	 $ git status
	 -------------------------------------------------------------------------------------------
	 to unstage a single file
	 $ git rm --cached filename.xyz
	 -------------------------------------------------------------------------------------------
	 to stage files by extension
	 $ git add *.xyz
	 and therefore, to unstage files by extension
	 $ git rm --cached *.xyz
	 -------------------------------------------------------------------------------------------
	 to add anything
	 $ git add . (yes, it's a dot)
	 -------------------------------------------------------------------------------------------
	 to commit changes
	 $ git commit -m "describe what you did" -a ('a' applies the comment to all committed files)
	 -------------------------------------------------------------------------------------------
	 to permanently exclude files from git add .
	 just create a file called .gitignore and fill it with what you want to exclude
 
	 
	   filename.xyz
	   /FolderOne
	   /FolderTwo/filename.xyz
	   *.xyz
	   /FolderThree/*.xyz
	   whatever.123
	  
	 -------------------------------------------------------------------------------------------
	 to list tracked files
	 git ls-tree -r master --name-only
	 -------------------------------------------------------------------------------------------
	 to create a new project's branch
	 $ git branch NewBranchName
	 -------------------------------------------------------------------------------------------
	 to switch to a specific branch
	 $ git checkout BranchName
	 -------------------------------------------------------------------------------------------
	 to switch back to the master branch
	 $ git checkout master
	 -------------------------------------------------------------------------------------------
	 to merge branch to master
	 $ git merge -m "describe what you're merging and convince me I should not terminate you right now"
	 -------------------------------------------------------------------------------------------
	 
	 
pushing stuff to some github account repositories


  	 -------------------------------------------------------------------------------------------
	 to return a list of remote repos
	 $ git remote
	 -------------------------------------------------------------------------------------------
	 to add a remote existing github repo (that someone already createds on github)
	 $ git remote add origin https://github.com/username/repositoryname.git
 	 -------------------------------------------------------------------------------------------
	 to push files to remote existing github repo (that you already added to project with git remote add)
	 $ git push -u origin master
 	 -------------------------------------------------------------------------------------------
	 to force push files to remote existing github repo
	 (this is not a good idea but it may be useful when pushing to github some already exixting project)
	 $ git push --force origin master
 	 -------------------------------------------------------------------------------------------
