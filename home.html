Git Command List


first you need to tell git that you want to use your current directory as a git environment.

user@host:~/path/to/project$ git init

output: Initialized empty Git repository in /home/path/to/project/


Once your have initialized your new empty repository, you can add your files.

The following will add all files and directories to your newly created repository:

user@host:~/path/to/project$ git add .

output: No output is good


when you want to "fix" modifies you need to "create a commit"

the commit must be marked with a comment which describes the modifies

user@host:~/path/to/project$ git commit -m "describe what you did" -a

output: (something similar)

[master (root-commit) 1b830f8] initial commit

0 files changed

create mode 100644 file


parameters:

-m, expects a commit message like: "describe what you did"

-a, applies the commit message to all the modified files
